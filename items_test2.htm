Item Area

Item Area reports the total surface area of straights and fittings, including seams.

(Note: Item Area may include “Skin” area for double wall duct.)

ü  The column name and justification may be modified in the Report Builder.

o   Users may specify the desired Units of measure.    (Selecting the “Show Units” check box will display unit designations in the column heading.)

o   Users may specify the minimum and maximum number of decimal places for the field.

o   Users may specify the preferred method of rounding.

o   Changing the Qty method will affect values reported in this field.

ü  Per Item Qty reports the same value (area including seams) whether there is one or multiple copies of the same item.

ü  By Item Rate reports the Area based on the “Cost Units” value, stored on the costing tab of the item.

·         The Costing Tab is located in the Item Properties Dialog.

·         In this case the Area is calculated as W x D x 12 because the Cost Units are set to per (ft).

·         Area values produced with this report field are adjusted for normalization (extra over).  Values do not represent the “True Area” bu the nominal area per (ft) for an ideally constructed straight.

·         The area for 1 quantity will be reported on items with Cost Units set to QTY.

ü  Total Item (Extn) reports the (area including seams ) x (value in the Quantity field).

Item From Catalog Flag

The Item From Catalog Flag reports a Yes/No value on the status of the Catalog checkbox of each item.

Catalogue

Catalog Items are templates that have been pre-configured to a single preset type. A catalog Item is designed to be placed on the drawing as it has been setup and only allows for basic Information to be changed as Takeoff. An example of this would be round duct fittings that can only be bought at a certain size. Catalog Items are less frequently used for this purpose as they do not allow the multiple Sizes to be set in the same Item. For this reason Product Lists were created. 

Product Lists

A Product list is a number of combined Catalog Items. Rather than have individual Items for each size in a range the Product list allows the user to have multiple preset sizes all contained in the same Item. This is used extensively in the Bought out Round Libraries.

ü  This field is useful for filtering or grouping cataloged into tables.

ü  The column name and justification may be modified in the Report Builder.
