How to set up your Outlook Signature...

  1. In Outlook, click on the Tools menu
  2. Click on Options (at the bottom)
  3. Choose the Mail Format tab
  4. Set Message Format to "Rich Text"
  5. Click on the Signatures button
  6. Click the New button
  7. Enter a name for the signature
  8. Check "Use file as template"
  9. Click Browse and navigate to f:\common\apps\
  10. Select outlook signature.rtf
  11. Click Next
  12. Edit the Name, Title, and direct telephone fields
  13. Click Finish
  14. Click OK
  15. In the Signature section at the bottom, pick your signature in both places
  16. Set Message Format back to HTML
  17. Click OK and you're done
